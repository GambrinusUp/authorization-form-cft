# Форма авторизации

## Инструкция по запуску:
1. Склонируйте репозиторий
```
git clone https://github.com/GambrinusUp/authorization-form-cft.git
```
2. Установите зависимости
```
npm install
```
3. Запустите
```
npm run dev
```
## Используемые технологии:
- React
- TypeScript
- Redux Tookit
- SCSS
- React Hook Form
- Vite
- Eslint
